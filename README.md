PJ Cipher 🔐
A sleek, cyberpunk-inspired Caesar Cipher web application built with React, featuring real-time encryption/decryption, binary stream visualization, and an interactive alphabet mapping circuit.

✨ Features
🔒 Encrypt and decrypt text using the Caesar Cipher
🎚️ Adjustable shift value (1–25)
⚡ Live conversion as you type
💻 Binary stream visualization of encrypted text
🧠 Interactive alphabet mapping circuit
📋 One-click copy to clipboard
🌧️ Animated Matrix-inspired background
🎨 Cyberpunk terminal-themed UI
📱 Responsive design for desktop and mobile

CSS3
